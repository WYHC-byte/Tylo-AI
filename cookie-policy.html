关于Cookies

最后更新：2024年8月6日

Cookie是存储在您浏览器中的小型文本文件，用于改善您的使用体验。 本政策说明了我们如何使用Cookie和类似技术。

我们使用的Cookie类型

必要Cookie 始终启用

这些Cookie对网站正常运行至关重要，无法在系统中关闭。

Cookie名称 用途 有效期
darkMode 保存界面主题偏好 永久
jiorkUserId 用户唯一标识 永久
jiorkLanguage 语言设置 永久
功能Cookie

这些Cookie使网站能够提供增强的功能和个性化。

Cookie名称 用途 有效期
jiorkChatHistory 保存聊天历史 永久
jiorkMemorySystem AI记忆功能 永久
jiorkUserNickname 用户昵称 永久
jiorkAchievements 成就系统数据 永久
分析Cookie

这些Cookie帮助我们了解访问者如何使用网站，以便改进服务。

目前我们不使用第三方分析Cookie。如果未来引入，我们会更新此政策并征求您的同意。

本地存储（LocalStorage）

除了Cookie，我们还使用浏览器的LocalStorage来存储数据。 LocalStorage与Cookie类似，但可以存储更多数据且不会随每个请求发送到服务器。

存储的数据

  • • 聊天历史记录
  • • AI记忆内容
  • • 用户设置和偏好
  • • 积分和等级信息

数据特点

  • • 仅存储在本地
  • • 不会发送到服务器
  • • 可随时清除
  • • 容量约5-10MB

如何管理Cookie和本地存储

通过我们的设置

点击本页面顶部的"管理Cookie偏好"按钮，选择您希望启用的Cookie类型。

通过浏览器设置

大多数浏览器允许您控制Cookie设置：

  • • Chrome: 设置 → 隐私和安全 → Cookie
  • • Firefox: 设置 → 隐私与安全 → Cookie和网站数据
  • • Safari: 偏好设置 → 隐私 → Cookie和网站数据
  • • Edge: 设置 → 隐私、搜索和服务 → Cookie

清除所有数据

您可以通过以下方式清除所有存储的数据：

  1. 进入JorkAI设置
  2. 点击"删除所有对话"
  3. 点击"清除所有记忆"
  4. 或直接清除浏览器缓存

第三方Cookie

我们使用的某些第三方服务可能会设置自己的Cookie：

支付服务

微信支付等支付平台可能设置Cookie以处理交易

CDN服务

Cloudflare等CDN可能设置安全相关的Cookie

我们无法控制第三方Cookie，建议查看相关服务的隐私政策。
